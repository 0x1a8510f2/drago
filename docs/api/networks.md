# Networks HTTP API

The `/networks/` endpoints are used to manage Networks.

## List Networks

This endpoint lists all Networks.

| **Method** |    **Path**     |    **Produces**    |
|------------|-----------------|--------------------|
|   `GET`    |   `/networks/`  | `application/json` |


| **ACL Required** |   
|-------------------------------------|
|   `management` for all policies.    |
